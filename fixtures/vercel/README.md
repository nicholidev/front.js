# Remix Vercel Fixture

```bash
# Install vercel globally
yarn global add vercel

# Make sure to build remix first, from the root of this repository
yarn && yarn build

# get into the Remix app
cd fixtures/vercel

# install deps
yarn

# link it to vercel
vercel link

# fire up the dev server
yarn dev

# open it up!
open http://localhost:3333
```
